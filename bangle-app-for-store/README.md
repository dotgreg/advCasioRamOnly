# Adv Casio Clock

An over-engineered clock inspired by Casio watches.
Forked from the awesome Cassio Watch

## Functionalities
- Current time
- Current day and month
- Footsteps
- Battery

- Simple Timer embedded  
- Weather forecast (7 days)
- Scratchpad
